# Project_Cataflix
Platform katalog film pribadi yang memungkinkan pengguna untuk mencari, mencatat, mengelola, dan meninjau film-film favorit mereka. Dengan integrasi ke database film global seperti TMDb, Cataflix memudahkan pengguna menemukan informasi lengkap tentang film, menyimpannya ke koleksi pribadi, serta memberi rating dan catatan secara personal.
